Auto Menu Settings
==================

Automates the creation of menu items for nodes by providing default options on
the content type's Configure form.

Installation
------------

- Install this module using the official Backdrop CMS instructions at
  https://backdropcms.org/guide/modules

- Edit a content type and set default values for menu items on the Menu Settings
  tab.
